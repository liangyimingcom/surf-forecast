# 移植功能 · 交互操作指南（surf-forecast）

> 分步操作：点哪里 → 看什么 → 预期结果。前端 = `web/浪报MVP.html`（移动端视图效果最佳）。
> 本地起服务：`SF_FRONTEND=web/浪报MVP.html PYTHONPATH=src uvicorn web.app:app --host 127.0.0.1 --port 8848` → 浏览器开 `http://127.0.0.1:8848`。

## A. 浪点收藏（R1.1）
1. 登录后，「📍 我的浪点」面板列出已保存浪点卡片。
2. 点卡片右侧 **☆** → 变 **★**，该浪点**置顶**；再点取消收藏。
3. 刷新页面 → 收藏状态保留（localStorage）。
- 预期：★ 收藏项始终排在前面；点击卡片主体切换到该浪点并加载浪报。

## B. 搜索 + 排序（R1.2）
1. 面板顶部搜索框输入浪点名称（如「流清河」）→ 列表实时过滤。
2. 排序下拉选择：收藏优先 / 名称 A→Z / 纬度北→南。
- 预期：无匹配时显示「无匹配浪点」；排序即时生效。

## C. 浪点地图（R1.3）
1. 点面板的「🗺️ 地图」按钮 → 展开地图（按钮变「🗺️ 收起」）。
2. 地图按真实经纬度标记所有浪点，自动聚焦。
3. 点标记 → popup 显示名称/坐标 + 「查看浪报」按钮 → 点它切换该浪点。
- 预期：标记数 = 有坐标的浪点数；点击「查看浪报」加载对应浪报。

## D. 公告详情（R2.1）
1. 「📢 公告」列表点任一条 → 富文本弹层打开。
2. 点 × / 背景 / 按 Esc → 关闭。

## E. 意见反馈（R2.2）
1. 「💬 意见反馈」选类型 + 填内容 → 点「提交反馈」。
2. 不选类型或空内容 → 红字报错；有效 → 绿字「✓ 已收到…（示例演示，未真实提交）」。
- 说明：演示环境不写库、不发网络请求。

## F. 关于·商务合作（R2.3）
- 「ℹ️ 关于 · 商务合作」查看关于我们/商务合作/联系方式（示例脱敏，占位邮箱 mailto 可点）。

## G. 活动墙（R2.4）
1. 「📰 活动墙」顶部类型 chips 切换（全部/浪报/攻略/活动/装备周边）→ 列表过滤。
2. 点活动卡片 → 富文本详情弹层（顶部「示例数据」badge）；Esc/背景关闭。

## H. 冲浪搭子/拼车（R2.5）
- 「🚗 冲浪搭子」只读列表：出发→到达 / 日期 / 余座 / 费用 / 备注 / 发布者 / 微信（**脱敏** `186****0095`）。不提供发布/编辑（合规）。

## I. 排水量计算器（R2.6，模块 F）
1. 「🏄 排水量计算器」输入体重（kg）+ 选水平（初学者…专业）→ 点「计算」。
2. 结果显示推荐排水量（L）。
- 校验：70kg + 中级 = **31.0 L**；70kg + 初学者 = **49.0 L**。

## J. 周边推荐（R2.7，模块 D）
- 「📍 周边推荐」按类型分组展示示例商户（冲浪店/餐厅酒吧/酒店）。

## K. 在线直播占位（R2.8，模块 A）
- 「📹 浪点直播」占位卡；点击弹出提示：本浪点暂无真实直播流（演示占位）。

## 常见问题
- **收藏面板不显示？** 需登录且已保存至少 1 个浪点（未登录/无浪点时自动隐藏）。
- **地图空白？** 离线或 Leaflet 未加载时按钮会提示；有网络会正常加载 OSM 瓦片。
- **反馈没真正提交？** 有意为之——演示环境不写库。
