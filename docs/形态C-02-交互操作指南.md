# 形态C整合 · 交互操作指南

> 前端 = 单 HTML `web/浪报MVP.html`（移动端视图最佳）。
> 本地起服务（含灌 58 浪点）：
> `SF_SEED_SPOTS=reference/data/shilaoren_spots.json SF_FRONTEND=web/浪报MVP.html PYTHONPATH=src uvicorn web.app:app --host 127.0.0.1 --port 8848`
> 浏览器开 `http://127.0.0.1:8848`（页面自动用 demo 账号建会话）。

## A. 浏览全国浪点目录
1. 页面加载后，「🌊 全国浪点目录」列出 58 浪点（登录态自动建立）。
2. 顶部**区域 chips**（全部/广东/海南/福建/广西/浙江/山东/国外/其他）→ 点击筛选。
3. 搜索框输入浪点名称或城市 → 实时过滤。
4. 有直播的浪点名称后有 **● LIVE** 标记；未校准朝向的标「朝向待校准」。
- 预期：右上角显示当前筛选浪点数；评分徽标随缓存/浏览出现（色阶）。

## B. 查看某浪点浪报（引擎自算）
1. 点击目录中任一浪点 → 页面滚到顶部，加载该浪点的**引擎浪报**。
2. 浪报含：综合评分、离岸风质、双周期 Tm/Tp、物理叙事、最佳窗口、板型、逐日图表、昨日回看。
- 预期：数据为 surf-forecast 引擎实算（GMT+8），非石老人预报值。

## C. 观看实时直播
1. 「📹 浪点直播」网格点任一卡片，或在浪报详情页点「📹 本浪点有实时直播」横幅。
2. 弹层用 hls.js 播放实时摄像头流（Safari 原生 HLS）。
3. 点 × / 背景 / Esc 关闭（自动停止播放、释放 hls 实例）。
- 说明：直播流来自公开来源，仅研究用途；需登录可见。

## D. 昨日回看校验
- 浪报页「昨日回看」区展示系统对**昨日**的预报（date 为 GMT+8 昨天）+ predict；切换浪点时随之更新。可对照体感自评。

## E. 收藏 / 搜索 / 地图（沿用）
- 「📍 我的浪点」面板：★ 收藏置顶、搜索、排序、🗺️ 地图（Leaflet，点标记切换浪点）。

## F. 社区/工具（示例）
- 公告详情 / 意见反馈 / 关于·商务合作 / 活动墙 / 拼车（脱敏）/ 排水量计算器 / 周边推荐 —— 均为示例数据，明确标注。

## 常见问题
- **目录/直播为空？** 需登录（demo 自动）+ 后端已灌注册表（`SF_SEED_SPOTS` 或生产 `tools/load_registry.py`）。
- **评分徽标不显示？** 本地无每日缓存时，点击浪点浏览后其评分回填；生产有缓存则批量显示。
- **直播卡顿/黑屏？** 上游直播流时效性问题，非本站逻辑；失效浪点会无卡片。
