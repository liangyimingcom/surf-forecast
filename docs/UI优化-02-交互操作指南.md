# UI 交互布局优化 · 02 交互操作指南

> 分步操作 + 预期结果。演示门禁口令 `surf2026`。移动端窄窗视图最佳。

## A 主标签页导航
- 顶部三个标签：**🌊 实时浪报**（默认）/ **📊 浪报详情** / **📋 其他**。
- 点击任一标签 → 仅显示该分区内容，导航**吸顶**始终可见。
- 预期：切换即时；页面滚动到顶部。

## B 实时浪报：目录 / 直播 子视图
1. 进入「实时浪报」默认显示 **🌊 全国目录**（58 浪点列表）。
2. 点子导航 **📹 浪点直播** → 切换到直播卡片网格，目录隐藏。
3. 点回 **🌊 全国目录** → 回到列表。
- 预期：两者互斥显示，避免一次性加载 ~100 卡。

## C 目录排序与搜索
1. 目录顶部：左搜索框、右**排序下拉**。
2. 选「综合评分 ↓」→ 列表按引擎评分高→低重排（无分的沉底）。
3. 选「有直播优先」→ 带 ● LIVE 的浪点排前。
4. 搜索框输入名称/城市 → 实时过滤；无结果显示「没有匹配的浪点…」。
- 预期：排序/搜索即时生效；空态有引导文案。

## D 深色模式
1. 右上角点 **🌙** → 整站切深色，按钮变 **☀️**。
2. 再点 **☀️** → 回浅色。
3. 刷新页面 → 保持你上次的选择（首次访问跟随系统深浅色）。
- 预期：卡片/输入/弹层均适配深色；选择持久化。

## E tab / 子视图记忆
1. 切到「其他」标签（或直播子视图）。
2. 刷新页面 → 自动回到「其他」（或直播子视图）。
- 预期：上下文不丢失；异常存储值安全回退默认。

## F 加载态
- 进入实时浪报瞬间：目录/直播区显示**骨架屏占位卡**。
- 切换浪点取报时：顶部出现**细 spinner 条**。
- 预期：无空白列表/白屏；数据到达后骨架消失。

## G 回到顶部
- 页面下滑超过一屏 → 右下角出现 **↑** 按钮，点击平滑回顶。

## 常见问题
- **看不到目录/直播/收藏**：需先通过门禁登录（口令 `surf2026`），未登录接口返回 401。
- **深色模式某处偏亮**：个别第三方嵌入（直播缩略图）不参与主题反色，属正常。
- **刷新回到实时浪报**：清除过站点数据或首次访问时按默认页显示。

## 第二轮操作（布局 + U-a~U-e）

### H 浪报详情新布局
- 浪点名「📍… ›」现在在**主标签栏下方**（点击跳浪报详情），旁边有 **🔗 分享** 按钮。
- 「📹 本浪点有实时直播」横幅在**日期评分条正下方**（本周必冲卡之前）。

### I 目录卡片直接收藏（U-a）
1. 实时浪报 → 全国目录，每张卡片右上角有 ☆/★。
2. 点 ☆ → 变 ★ 收藏（不会跳转到该浪点浪报）；再点取消。
3. 收藏与「浪报详情」收藏面板、地图着色同步。

### J 地图收藏着色（U-b）
- 浪报详情 → 我的浪点 → 🗺️ 地图：**金色标记=已收藏**、蓝色=未收藏，底部有图例。

### K 键盘 / 无障碍（U-c）
- Tab 键在标签/按钮间移动会显示蓝色焦点框；主标签对读屏宣告为 tablist/tab 选中态。

### L 移动端 chips 横滑（U-d）
- 区域筛选、活动类型 chips 过多时可**左右横滑**（不再换行占多行）。

### M 分享浪点（U-e）
1. 浪报详情 → 浪点名旁点 **🔗 分享** → 链接已复制（形如 `…#spot=36.09,120.47,青岛山东头`）。
2. 把链接发给别人/自己，打开即**自动定位该浪点并进浪报详情**。

## 第三轮操作（V-a~V-e）

### N 目录「仅看直播」（V-a）
- 实时浪报 → 全国目录工具条 → 点 **📹 仅直播**（变红=开启）→ 只显示带 ● LIVE 的浪点；再点关闭。可与区域 chips/搜索/排序同时用。

### O 目录「收藏优先」排序（V-b）
- 目录排序下拉选 **收藏优先** → 已 ★ 收藏的浪点排到最前。

### P 详情锚点快捷跳转（V-c）
- 浪报详情页浪点名条下方一排：**📊 评分 / 📈 图表 / 🔁 昨日回看 / 📹 直播**，点击平滑滚动到对应区块。

### Q 首访引导（V-e）
- 首次打开顶部出现引导横幅，读完点 **知道了 ✕** 关闭，之后不再出现。
