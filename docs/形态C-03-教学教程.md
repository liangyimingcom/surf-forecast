# 形态C整合 · 教学教程（开发者）

> 如何在 surf-forecast 统一后端下整合外部浪点源 + 直播，复用引擎评分。含架构、代码地图、扩展模式、红线、测试。

## 一、5 分钟上手
```bash
cd surf-forecast-kiro-v2 && . .venv/bin/activate
# 1) 一次性导入石老人 58 浪点快照(需上游可达)
python tools/import_shilaoren_spots.py           # → reference/data/shilaoren_spots.json
# 2) 本地起服务(灌注册表)
SF_SEED_SPOTS=reference/data/shilaoren_spots.json SF_FRONTEND=web/浪报MVP.html \
  PYTHONPATH=src uvicorn web.app:app --host 127.0.0.1 --port 8848
# 3) 测试
pytest -q                                        # 126 passed
node web/e2e/new_features.mjs http://127.0.0.1:8848   # 30/30
# 4) 生产灌注册表(一次性)
SF_STORE=dynamo SF_TABLE_PREFIX=surf-forecast-dev AWS_PROFILE=oversea1 \
  PYTHONPATH=src python tools/load_registry.py reference/data/shilaoren_spots.json
```

## 二、架构（形态C = 统一后端）
```
外部源(石老人 isurf, 仅研究)          surf-forecast 统一后端(FastAPI)
  getCamera/getNewForecast  ──导入──▶ spot_registry(DynamoDB, 58浪点+live_src+facing)
  isurfvideo HLS ◀──前端直连(hls.js)   /api/catalog  目录(58)
                                        /api/cams     直播源(含live_src)
                                        /api/catalog/scores 评分(缓存回批)
                                        /api/report   引擎自算(评分/离岸风/双周期/wdeg)
                                        /api/report/history 昨日回看
前端 web/浪报MVP.html: 目录/直播/浪报/收藏/地图/社区示例
```
**原则**：预报统一走引擎；直播视频前端直连(不经后端代理)；石老人上游仅一次性导入坐标+live_src。

## 四、如何接入一个新的外部浪点源（模式）
1. 写导入脚本 → 规整为 {slug, name, lat, lon, facing, live_src, region_cn} 快照 JSON。
2. `seed.build_registry_rows` 复用（或扩展字段），`seed_store` 写注册表（**必过 upsert_registry → _to_decimal**）。
3. 预报**不接外部值**，一律 `/api/report` 引擎自算（保证 wdeg/双周期/GMT+8 契约一致）。
4. 直播/媒体前端直连（CORS 允许时），不经后端代理。
5. 新增只读路由复用 `Depends(deps.current_user)`（401 门禁）。
6. 加 pytest 契约 + 扩 `web/e2e/new_features.mjs`；`node --check` + `pytest` + E2E。

## 五、踩坑（本次真实经历）
1. **并发写冲突**：曾两个 Goal 驱动器同改文件致 tasks 与代码失同步 → 单一驱动器 + 改前 grep 实际文件 + 勾选须与文件一致。
2. **浪点朝向 facing**：石老人无此字段，离岸风质依赖 → 区域粗估 + 标「待校准」。
3. **58×实时开销**：目录评分用缓存回批（/api/catalog/scores），本地无缓存返回 cached=false + 点击回填，避免 58 次实算。
4. **hls.js 隐藏 video**：关闭弹层须 destroy hls 实例 + pause/removeAttribute('src') 释放。
5. **过时 E2E 断言**：占位升级为真实功能后，旧断言(占位===4)需同步改为真实(≥10)。

## 六、红线与合规
- GMT+8 / wdeg / 双周期；DynamoDB 写过 float→Decimal；**不改引擎内核**；受保护接口 401；ALB SG 禁 0.0.0.0/0；terraform 禁 -auto-approve。
- 石老人逆向仅研究用途；直播 401 门禁 + 来源免责；社区示例；不做登录/支付/写入。
